# Сервис с авиабилетами

## Структура приложения:

[Feature Sliced Design](https://feature-sliced.design/)

## Стек технологий:
    - React
    - TypeScript
    - RTK Query
    - Redux
    - Styled-components
    - Express server

Для запуска приложения и сервера на Node.js требуется:

### `npm install`

### `npm run start`
### `cd src/server && node server.js`
